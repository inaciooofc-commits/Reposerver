# 🎵 Anime Pulse Music Server

**Servidor de música compartilhada com AntiX Panel**

## Instalação no Antix Linux

```bash
git clone https://github.com/inaciooofc-commits/Reposerver.git
cd Reposerver
chmod +x install_antix.sh
./install_antix.sh
```

Acesse: http://localhost:5000

## Comandos
- `python3 reposerver_main.py` - Iniciar servidor
- `./install_antix.sh` - Instalação otimizada para Antix
